# 🚀 Automação Python + Google Sheets API

Este projeto demonstra como integrar o Python com o Google Sheets para criar um sistema de log de mensagens. Ele utiliza o Google Apps Script para transformar uma planilha em uma API (Web App) capaz de receber e enviar dados.

## 🛠️ Tecnologias Utilizadas

*   **Python 3.x**: Linguagem principal.
*   **Pandas**: Para manipulação de dados.
*   **Requests**: Para comunicação com a API (métodos GET e POST).
*   **Pytest**: Para garantir a qualidade e funcionamento do código através de testes unitários.
*   **Google Apps Script**: Ponte entre a planilha e a internet.

## 📋 Funcionalidades

*   Envio de mensagens individuais e em massa para a planilha.
*   Recuperação de todos os registros salvos no Google Sheets.
*   Busca automática da última mensagem registrada.
